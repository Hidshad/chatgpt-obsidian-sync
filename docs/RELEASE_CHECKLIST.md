# Release Checklist

Checklist for `v0.1.0-preview`.

- [ ] Start the local service successfully.
- [ ] Load the browser extension successfully.
- [ ] Confirm non-conversation ChatGPT pages stay quiet.
- [ ] Export a short current conversation successfully.
- [ ] Export a long current conversation successfully.
- [ ] Confirm part files are continuous.
- [ ] Confirm exported section count equals database message count.
- [ ] Confirm order issue count is 0.
- [ ] Realtime sync a User message successfully.
- [ ] Realtime sync an Assistant message successfully.
- [ ] Stop realtime sync and confirm there are no further `POST /api/messages` calls.
- [ ] Regenerate current conversation notes successfully.
- [ ] Confirm background logs show no unexpected 400/500 errors.
- [ ] Confirm the extension does not save ChatGPT token/cookie.
- [ ] Confirm README is updated.
- [ ] Confirm CHANGELOG is updated.
- [ ] Confirm known limitations are documented.
